# PRODDEC Community Forum

A responsive community forum frontend built with **React + Vite**.

## Features

- 📌 **Forum page** with posts/threads, pinned posts, and pagination-ready list
- 🔍 **Real-time search** across post titles, bodies, tags, and authors
- 🏷️ **Category filters** (General, Showcase, Help & Support, Feedback, Ideas, Jobs)
- 🔃 **Sort options** – Newest, Popular, Most Replied
- ✏️ **Create post modal** with form validation
- ❤️ **Like / unlike** posts with optimistic UI updates
- 📱 **Responsive design** – hamburger menu on mobile, fluid layouts
- 🎨 **Clean UI** with gradient hero, stats bar, smooth hover effects

## Tech Stack

- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)
- Plain CSS (no UI library dependencies)

## Getting Started

```bash
npm install
npm run dev       # development server
npm run build     # production build
npm run preview   # preview production build
```
